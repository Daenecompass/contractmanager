# ContractManager

ContractManager is a contract management software for the Jameica platform (http://willuhn.de/products/jameica/). It allows you to enter details of your contracts, e.g., financial information, runtime, or cancellation deadlines. It notifies you about upcoming cancellation deadlines.

For installation instruction, see http://janrieke.de/contractmanager/.

[ContractManager Web Site](http://janrieke.de/contractmanager/)

## Features
* Management of contractual details, e.g., financial information, addresses, terms, documents
* Reminders for cancellation deadlines
* Export reminders to iCalendar file
* Generation of letters of cancellation
